Installation

It's really simple.

Here are 2 ways to upload the theme. Please fallow the given steps.

Via Dashboard

  1. Navigate to Appearance → Themes in your WordPress admin dashboard.
  2. Click the Add New button at the top of the page then go for the Theme Upload option.
  3. For the file upload, pick Theme Files / emyui.zip in the theme package downloaded from ThemeForest and click Install Now.
  4. Click Activate once the upload has finished and you're ready for the next step: Installing the Plugins.

 

Via FTP

  1. Extract the theme package downloaded from ThemeForest emyui-wp-full.zip open the folder.
  2. Open any FTP aplication like Filezilla navigate to the directory of the Theme.
  3. For the file upload, pick Theme Files / emyui.zip in the theme package downloaded from ThemeForest.
  4. Navigate to Appearance → Themes in your WordPress admin dashboard.
  5. Click Activate once the upload has finished and you're ready for the next step: Installing the Plugins.

Demo import & Plugins

One Click Installation

Emyui comes with one click demo installation exactly like the original demo. You just need to verify that you have set the

  • Memory Limit: 256 MB
  • Max. Execution Time: 0 (Unlimited)
  • Max. Post Size: 256 MB

After theme activate, navigate to appereance > install plugin . then you must install and activate all necessary plugins.

after all your plugins has been activated navigate to appereance > import demo data. and follow steps to import the demo.

Homepage & Blog

After demo import it's required to do this step too.

To setup the home page, in your WordPress admin panel, navigate to Settings → Reading.

Assuming you already have a page named "Front Page", for the Front page display, select "Home" and choose your "Homepage one" page as the home page. This is what you should be seeing on your screen from left side.

 

To setup the blog page, in your WordPress admin panel, navigate to Settings → Reading.

Assuming you already have a page named "Blog", for the Front page display, select "A static page" and choose your "Blog" page as the Posts page. This is what you should be seeing on your screen from left side.

 

Menus Setup

After installation the Emyui and after dummy-content import, you need to select our main menu from WordPress admin → menus → theme locations.

Also You need to select the footer menus one-by-one. for doing that create 3 different menus and assing them to footer 1 to 3 menus location.

 

Header

In order to customize the header section you must navigate to Dashboard > Boostify header footer builder 

here you will see different templates which are the mega menus that you can customize using elementor. and also by customizing the main-header template you can control the entire header.

 

 

 

Footer

for footer section you must navigate to Dashboard > Emyui Settings > Footer Section.

Note: the footer menus can be changed by menu setup step guide.
